Aplicación en tiempo real

Modulr utiliza websockets para mantener conexiones en tiempo real entre clientes y servidor.

Un solo lenguaje

Modulr utiliza javascript para todo el desarrollo de las tecnologías que incorpora.

Utiliza los Gestores de base de datos que prefieras

Modular trabaja con múltiples gestores de bases de datos. Es parte de nuestra libertad.

Fácil de usar

Hasta el momento

modulr esta compuesto por los siguientes módulos

  • 1

    Login, Registo y Recuperación de Passwords

    Registrate, inicia de sesión y recupera passwords extraviados.

  • 2

    Usuarios y Control de Acceso

    Alta, baja y asignación de permisos a los usuarios del sistema.

  • 3

    Bitácoras de Acceso

    Quién entró, cuándo, cuánto tiempo y desde dónde.

  • 4

    Perfil de Usuario

    Edita información de los usuarios tanto personal como de la compañía.

  • 5

    Chat

    Chat en tiempo real entre los empleados y sus departamentos.
